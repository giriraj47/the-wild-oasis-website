# The Wild Oasis Website

The Wild Oasis is a modern cabin booking web application built using **Next.js**, **Tailwind CSS**, and **Supabase**. The project focuses on providing a clean user experience for browsing cabins, checking availability, and making reservations, while leveraging server-side rendering and modern React patterns.

## Demo

Live demo :

https://the-wild-oasis-website-theta-one.vercel.app/

## Features

- Browse available cabins with images, pricing, and capacity details  
- View individual cabin pages with detailed descriptions  
- Date-based booking system  
- User authentication and session handling  
- User-specific bookings and protected routes  
- Responsive UI for mobile and desktop  
- SEO-friendly pages using Next.js App Router  

## Tech Stack

- **Next.js** – React framework with App Router and server components  
- **React** – Component-based UI  
- **Tailwind CSS** – Utility-first styling  
- **Supabase** – Backend for database, authentication, and storage  
- **Auth.js / NextAuth** – Authentication (if configured)
